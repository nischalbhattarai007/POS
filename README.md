# POS

## Tech Stack

### Frontend
- **HTML5** - Markup
- **CSS3** - Styling
- **JavaScript (Vanilla)** - Client-side logic

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MySQL** - Database

## Features

- User Authentication (Superadmin & Admin roles)
- Product Management with Categories
- Customer Management
- Invoice & Billing System
- Company Settings
- User Management
- Role-based Access Control
- Product Upload Support
- Dashboard & Reports

## Installation

### Backend Setup
```bash
cd pos-backend
npm install
# Configure database in src/config/database.js
npm start
